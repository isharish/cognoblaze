Cognoblaze'23
Cognoblaze is a national level technical symposium that aims to provide opportunity to each and everyone,to improve their expertise and adroitness in various subjects. Taking part in the technical events encourage innovative and critical thinking as well as non technical event, improve creative thinking by thinking outside the box. Conclusively, these events promote the development of various skills with relevance to our presence of mind, presentation skills and fostering our talents to a great extent.
Technical Events

EXPO CLASH

EXPO CLASH

Expo clash is a paper presentation event where a team of two or three will come up with a recent technical topic ( In the domains like : Artificial intelligence , Cloud computing, Augmented reality, Computer security, Machine learning, Data Science etc.) and deliver it in detail using PowerPoint presentation.

PowerPoint Presentations will be collected in prior. Those PPTs which qualify will be presented on the day of the event.

SWACHH CODE

SWACHH CODE

Swachh code is a bug clearance and execution of the program to get the desirable output while the code snippet with infused bugs will be given.

SUPERFICIO

SUPERFICIO

Superficio is a frontend development event where html/css code for the given design must be written to replicate the design.

JACK OF ALL TRADES

JACK OF ALL TRADES

Jack of all trades is an all in one event which includes MCQs of all core subjects of Computer science.

BLANKY BLANK

BLANKY BLANK

Blanky blank is a word game where the team of three is supposed to find a word for which there will be 3 clues which will be revealed in auction basis i.e, the team who bids the maximum value of currency will get to know the clue. The team which finds the maximum words using less currency will be the winner.

( Individuals can register for this event . You will be joined with the teams on the day of the event. If you have formed a team already, register it individually for now and team mates can join together on the day of the event.)

Non-Technical Events

CREATOR'S CORNER

CREATOR'S CORNER

Creator's corner is an event of logo creation for the given description using pencil and paper or any known tools.

REPLICOLOGY

REPLICOLOGY

Replicology is an event of working in MS Word where an image of a book page will be shown and it has to be replicated in the word file.

THE INTELLECT

THE INTELLECT

The Intellect is an event with all General knowledge MCQs.

MEMORY DUMP

MEMORY DUMP

Memory dump is a memory game which solely depends on remembering the numericals.
